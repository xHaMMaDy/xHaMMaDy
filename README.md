<h3>👋 &nbsp; About me</h3>

I'm a **bioinformatics engineer** working at the seam where **single-cell genomics**, **cheminformatics**, and **machine learning** meet. I build reproducible pipelines that move ideas from raw biology to deployable software — APIs, Docker images, fairness audits, and all.
