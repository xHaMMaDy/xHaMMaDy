### 🧬 About

🎓 Bioinformatics @ **Nile University** &nbsp;•&nbsp; 📍 Egypt
🔬 **Research interests:** Alzheimer's disease &nbsp;•&nbsp; single-cell transcriptomics &nbsp;•&nbsp; QSAR & molecular docking &nbsp;•&nbsp; EHR machine learning
🛠️ I build reproducible pipelines that bridge **omics, cheminformatics, and ML**.

🚀 **Currently working on**
- 🧠 [**AD-scRNA2QSAR**](https://github.com/xHaMMaDy/AD-scRNA2QSAR) — scRNA-seq → drug-target → QSAR → Flask API
- 🏥 [**readmit-bench**](https://github.com/xHaMMaDy/readmit-bench) — calibrated XGBoost + FastAPI + Flask SaaS dashboard
- 💊 **MyPaperDocking** — large-scale virtual screening across 5 protein targets
